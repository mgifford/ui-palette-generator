UI Palette Generator - Hash Format Test

New Hash Format (Compact)

Example with seed color and harmony mode:

http://localhost:8000/#seed=A7C39F,harm=split-complementary

Total characters: 60

Old Hash Format (Verbose - Deprecated)

Example with same configuration:

http://localhost:5000/#colors=light.colors=LIGHT.COLORS=LIGHT.COLORS=LIGHT.COLORS=LIGHT.COLORS=LIGHT.COLORS=LIGHT.COLORS=LIGHT.COLORS=ACCENT=E960B7,accent=A7C39F,harmony=split-complementary

Total characters: 200+ (much longer)

Hash Parameters

Parameter Example Description
seed seed=A7C39F Seed color (hex or color name)
harm harm=split-complementary Harmony mode (analogous, triadic, split-complementary, etc.)
l[token] lcanv=E5F6CA Light theme color override (l=light, token=abbreviated name)
d[token] dcard=313230 Dark theme color override (d=dark, token=abbreviated name)

Token Abbreviations

Used with theme prefix (l=light, d=dark):

Short Full Name Category
canv canvas Background
crd card Background
ans accentNonContentStrong Accent
ansub accentNonContentSubdued Accent
